"Table Habit" is an app that helps you establish and track your own micro habit. Its includes a complete set of growth curves and charts to help you establish habits more effectively.

Key Features:
• Scoring system to help you develop your own micro habits
• Supports both positive and negative habits
• Easy-to-use check-in UI
• A quick search and filter to help quickly find past habits.
• Color-coded habits for quick visual distinction
• Import and export habits using a human-readable JSON format
• Optimized for landscape and large-screen devices
• WebDAV sync support for cross-device syncing

Moreover, this app completely open source.
